About

KYB-Donut, OCR-free KYB extraction
Donut · FastAPI · Celery · React
Document AI
Problem
Indian fintech onboarding pulls structured fields from skewed scans of GST, PAN, and partnership docs, OCR + regex fails silently and manual review is expensive.
Product
OCR-free document understanding pipeline with a reviewer queue for low-confidence cases and an audit trail for compliance.
Approach
Donut encoder–decoder model, FastAPI + Celery async workers, synthetic data generation, threshold-routed HITL queue, React reviewer UI.
